Installation

  1. Unpack the package files into a directory.
  2. Start Delphi IDE.
  3. Open Packages\dklangN.dpk (where N is your Delphi version: 12 for BDS 2009) and click Compile.
  4. Open Packages\dcldklangN.dpk (where N is again your Delphi version), click Compile, then click Install.
  5. Close the files (don't save the changes if any).
  6. Add the path to DKLang.pas to the IDE library path.

Notice that both compiled packages (dklangN.bpl and dcldklangN.bpl) must reside in a directory listed in system PATH (BDS menu: Tools|Options... then Environment Options|Delphi Options|Library - Win32 then Library path).

If you're upgrading: don't forget to read Upgrading from previous versions section.
